# Product Inventory App (Laravel + AJAX)

##  Overview
A simple Laravel-based product inventory system that allows users to manage products without page reload using AJAX.  
Data is stored in a JSON file instead of a database.

---

##  Features

- Add products via AJAX
- Edit products without page refresh
- Delete products instantly
- Store data in JSON file (`storage/app/products.json`)
- Display live-updating product table
- Automatically calculate total value per product
- Display sum of all products

---

##  Tech Stack

- Laravel
- PHP
- JavaScript (Fetch API / AJAX)
- Bootstrap 5
- JSON file storage
